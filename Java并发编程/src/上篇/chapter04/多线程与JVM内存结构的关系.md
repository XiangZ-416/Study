# 多线程与JVM内存结构的关系

## JVM内存结构
   结构：栈 + 堆
   
   栈：方法区、虚拟机栈、本地方法栈、程序计数器
        方法区：存放方法的入口、出口；类加载的信息（常量、静态变量等）；线程共享。
        虚拟机栈：每一个线程执行时都会创建栈帧（栈帧中存放存储局部变量表、操作数栈、动态链接库（方法入口）、方法出口）；线程私有。
        本地方法栈：C++写的方法。如start0方法。线程私有。
        程序计数器：字节码行号指示器。线程私有。
   堆：存放对象实例、数据实例、垃圾收集器主要的管理区域。线程共享。
        执行引擎：将字节码文件翻译成操作系统能认识的机器指令。
