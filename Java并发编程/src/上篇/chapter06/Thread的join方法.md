# Thread的join方法及案例

## 一、join()方法
    1.join()：如t1.join()的含义是当前线程等待t1线程执行结束后再执行;
    2.join(long millis)：如t1.join(100)的含义是当前线程等待t1线程100ms，超过100ms t1线程如果还没结束，当前线程会执行。
                                                    如果t1是非守护线程，那么当前线程执行结束后，t1继续执行；否则不再执行t1线程。
    注意：join()必须放在start()之后才会起作用。
